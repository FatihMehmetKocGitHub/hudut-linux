# Hudut Tor Kullanım Kararı

Hudut Linux V1 içinde Tor Browser ve Tor Research profili bulunur.

Ancak Hudut Linux V1'in amacı aktif ve hassas Tor operasyon sistemi olmak değildir.

## Resmi Karar

Hudut Linux V1 içinde Tor:

- Demo
- Eğitim
- Temel bağlantı testi
- Shield profil gösterimi
- Tor Research workflow anlatımı

amacıyla bulunur.

Aktif ve hassas Tor araştırmaları için ayrı USB tabanlı Tails kullanımı önerilir.

## Neden?

Hudut Linux geliştirme sisteminde kişisel kullanım izleri bulunabilir:

- GitHub hesabı
- Normal Firefox profili
- ChatGPT oturumu
- Geliştirme dosyaları
- Yerel raporlar
- Kişisel bağlantılar
- Welcome Center içindeki sosyal bağlantılar

Bu nedenle Hudut installed sistem, hassas Tor çalışmaları için ana ortam olarak konumlandırılmaz.

## Hudut İçinde Tor'un Rolü

Hudut içinde Tor Browser:

- Tor Research profilini göstermek
- check.torproject.org ile bağlantı doğrulama göstermek
- Eğitim videosunda Tor workflow anlatmak
- Yasal OSINT sürecinde not alma akışını göstermek

için kullanılabilir.

## Tails Kullanım Rolü

Tails USB:

- Aktif Tor araştırması
- İz bırakmadan geçici çalışma
- Kişisel sistemden ayrılmış kullanım
- Hassas kaynak inceleme

için tercih edilir.

## Hudut İlkesi

Hudut Linux V1:

- Secure OSINT Workstation
- CTI / GEOINT / SOCMINT araştırma ve raporlama ortamı
- Eğitim ve portföy projesi

olarak konumlandırılır.

Hudut Linux V1, Whonix veya Tails alternatifi olarak pazarlanmaz.

## Güvenlik Notu

Tor Browser kullanırken:

- Kişisel hesaplara giriş yapılmaz
- Eklenti kurulmaz
- Torrent kullanılmaz
- İndirilen dosyalar ana sistemde açılmaz
- Normal Firefox ile aynı araştırma sürdürülmez
