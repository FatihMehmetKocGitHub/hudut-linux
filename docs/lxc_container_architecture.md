# LXC / LXD Container Mimarisi

Hudut Linux V1'de OSINT, CTI, TECHINT, geliştirme ve honeypot işlemleri mümkün olduğunca izole container ortamlarında yürütülecektir.

## Planlanan Containerlar

- hudut-osint
- hudut-cti
- hudut-techint
- hudut-dev
- hudut-sandbox
- honeypot-lab

## Temel Mantık

Ana sistem günlük kullanım ve yönetim için temiz kalır.
