# Hudut Linux V1

**Hudut Linux V1**, Xubuntu 24.04 tabanlı güvenli OSINT ve siber istihbarat çalışma istasyonu projesidir.

Proje; yasal açık kaynak araştırması, CTI, TECHINT, GEOINT/IMINT, raporlama süreçleri ve izole LXC laboratuvarı için tasarlanmıştır.

> Hudut Linux bir saldırı platformu değildir. Savunma, araştırma, analiz, dokümantasyon ve raporlama odaklıdır.

## Güncel Durum

- Sürüm: `0.4.2-public-showcase-alpha`

- Durum: Alpha
