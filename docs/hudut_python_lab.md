# Hudut Python Lab

Hudut Python Lab, Hudut Linux içinde Python geliştirme, AI denemeleri, otomasyon scriptleri, terminal araçları ve masaüstü Python uygulamaları için ayrılmış özel çalışma alanıdır.

## Ne İşe Yarar?

Hudut Python Lab şu işler için kullanılır:

- Python öğrenme ve pratik
- AI ve veri analizi denemeleri
- OSINT otomasyon scriptleri
- Terminal tabanlı yardımcı araçlar
- Legal lab ve analiz scriptleri
- Masaüstü Python uygulamaları
- Raporlama ve veri işleme

## Ana Klasör

~/hudut-python-lab

## Python Sanal Ortamı

~/hudut-python-lab/.venv

Bu sanal ortam sayesinde Python paketleri sistem Python ortamına karışmaz.

## Kullanım

Aktif etmek için:

cd ~/hudut-python-lab
source .venv/bin/activate

Kapatmak için:

deactivate
