# Hudut Linux 0.1 Beta Release Notes

## Release

Hudut Linux 0.1 Beta is an early developer beta of the Hudut Linux Secure OSINT Workstation project.

## Purpose

This release provides a Xubuntu-based secure research workstation layer for legal OSINT, CTI, GEOINT, SOCMINT, reporting and controlled research workflows.

## Included

- Hudut Shield Security Center
- Security profile system
- Kernel hardening checks
- Proxy Manager
- Firefox OSINT profile
- Tor Research workflow
- Tor service controls
- Desktop launchers
- Welcome Center
- Calamares installer demo branding
- Penguins' eggs ISO/remaster tooling
- Documentation and ethical usage policy

## Not Included Yet

- Final ISO installer flow
- Full Calamares installation modules
- Separate hudut research user
- LXC/LXD lab environment
- Tool installer
- Automated OSINT tooling
- Final 1.0 ISO release

## Legal Notice

Hudut Linux is not an attack system. It is developed for legal open-source intelligence, education, reporting and official notification workflows.
