# Resmi Bildirim Kanalları

Hudut Linux ile yapılan araştırmalarda tespit edilen riskler sorumlu ve yasal biçimde bildirilmelidir.

## Bildirim Mantığı

1. Bulguyu doğrula
2. Kanıtları düzenle
3. Kişisel verileri gereksiz şekilde paylaşma
4. Teknik rapor hazırla
5. Yetkili kuruma veya ilgili güvenlik ekibine bildir
6. Kamuya açık ifşa yapma

## Türkiye İçin Örnek Kanallar

- USOM
- EGM Siber Suçlarla Mücadele
- Jandarma Siber Suçlarla Mücadele
- KVKK ihlal bildirimleri
- CİMER
- İlgili kurumun resmi güvenlik / abuse / iletişim adresleri

## Ana Kural

Şov yapma. İfşa etme. Resmi kanala ilet.
