# Hudut Linux Kurulum Rehberi

Hudut Linux V1, Xubuntu tabanı üzerine geliştirilen yasal OSINT, CTI ve GEOINT araştırma ortamıdır.

## İlk Kurulum Sırası

1. Xubuntu kurulumu
2. Sistem güncellemesi
3. Temel paketlerin kurulumu
4. GitHub SSH bağlantısı
5. Hudut Linux reposunun klonlanması
6. Logo, wallpaper, GRUB, login ve Plymouth ayarları
7. Welcome Center kurulumu
8. Hudut Shield Security Center kurulumu
9. LXC/LXD laboratuvar ortamlarının kurulumu
10. OSINT/CTI araç modüllerinin eklenmesi
