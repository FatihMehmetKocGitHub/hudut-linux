# Hudut Control Panel

Hudut Control Panel, Hudut Linux V1 içindeki ana modülleri tek merkezden açmak ve temel durumları kontrol etmek için hazırlanmıştır.

## Bağlanan Modüller

- Hudut Shield
- Hudut LXC Manager
- Hudut Tool Installer
- Proxy Manager
- LXC Status Report
- Git Status
- Version Info
- Proje / Docs / Reports / Intelligence / Tools klasörleri

## Amaç

Hudut Linux V1 içinde komut satırına dağılmış yönetim işlerini tek GUI panel altında toplamaktır.

## Güvenlik Çizgisi

Hudut Linux V1 saldırı sistemi değildir. Yasal OSINT, CTI, TECHINT, GEOINT, güvenli araştırma ve raporlama workstation katmanıdır.
