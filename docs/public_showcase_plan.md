# Hudut Linux V1 Public Showcase Plan

## Amaç

Private geliştirme reposundaki güvenli ve tanıtıma uygun dosyaları temiz bir public vitrin olarak hazırlamak.

## Public Repo İçeriği

- PUBLIC_README.md
- version.json
- docs/
- güvenli uygulama kaynakları
- intelligence README/checklist/source/report şablonları
- güvenli örnek raporlar
- launchers/
- seçilmiş LXC scriptleri
- seçilmiş installer scriptleri

## Public Repo Dışında Kalacaklar

- `.git/`
- yerel raporlar
- loglar
- geçici dosyalar
- cache dosyaları
- browser profilleri
- LXC container verileri
- gerçek araştırma çıktıları
- kimlik bilgileri ve tokenlar

## Yayın Akışı

1. Private repo temizliği
2. Public preview üretimi
3. Hassas veri taraması
4. Manuel inceleme
5. Public repo push
6. Alpha release etiketi
