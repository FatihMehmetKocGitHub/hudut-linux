# Araç Kullanım Politikası

Hudut Linux içindeki araçlar yalnızca yasal, etik ve izinli araştırmalar için kullanılmalıdır.

## İzinli Kullanım

- Açık kaynak araştırma
- Kendi sistemlerinde test
- Yazılı izinli güvenlik analizi
- CTI ve IOC incelemesi
- GEOINT / IMINT doğrulama
- Raporlama ve farkındalık çalışmaları

## Yasak Kullanım

- İzinsiz erişim
- Yetkisiz ağ taraması
- Doxxing
- Kişisel veri yayma
- Kimlik avı saldırısı
- Zararlı yazılım geliştirme
- Başkasına ait sistemlerde izinsiz test

## Temel İlke

Bul, doğrula, analiz et, raporla ve resmi kanala ilet.
