# CRISIS INTEL Checklist

## Hazırlık

- [ ] Araştırma amacı açık mı?
- [ ] Kapsam ve sınır belirlendi mi?
- [ ] Yasal / etik sınır kontrol edildi mi?
- [ ] Kişisel veri riski değerlendirildi mi?
- [ ] Kullanılacak kaynaklar listelendi mi?

## Toplama

- [ ] Kaynak adı kaydedildi
- [ ] URL / referans kaydedildi
- [ ] Erişim tarihi kaydedildi
- [ ] Ekran görüntüsü veya not alındı
- [ ] Kaynak güvenilirliği değerlendirildi

## Analiz

- [ ] Bulgular sınıflandırıldı
- [ ] Tek kaynaklı iddialar ayrıca işaretlendi
- [ ] Zaman çizelgesi çıkarıldı
- [ ] Teknik göstergeler ayrıldı
- [ ] Yanlış pozitif ihtimali değerlendirildi

## Raporlama

- [ ] Yönetici özeti yazıldı
- [ ] Bulgular kanıtla eşleştirildi
- [ ] Risk seviyesi belirtildi
- [ ] Öneriler eklendi
- [ ] Etik / yasal not eklendi
